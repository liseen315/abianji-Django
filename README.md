# abianji
Abianji is a blog system with Django

## datebase

```
$ pipenv install mysqlclient
```

## Installation
```
$ git clone git@github.com:liseen315/abianji.git
$ cd abianji
$ pipenv install
$ pipenv shell
$ python manage.py runserver 
```
